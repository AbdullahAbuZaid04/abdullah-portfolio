# Portfolio

مرحباً! هذا هو الكود المصدري لموقع أعمالي الشخصي، تم بناؤه ليكون سريعاً، تفاعلياً، ومتوافقاً مع كافة الشاشات.

---

## التقنيات المستخدمة
- **React.js**: لبناء واجهة المستخدم.
- **Tailwind CSS**: لتنسيق التصميم بسرعة وكفاءة.
- **GSAP**: لإضافة حركات (Animations) سلسة واحترافية.

---

## المميزات
- **Animations**: حركات تفاعلية متميزة باستخدام GSAP.
- **Responsive**: تصميم مرن يتناسب مع الجوال والحاسوب.
- **Modern UI**: واجهة بسيطة وعصرية باستخدام Tailwind.

---

##  تشغيل المشروع

1. تحميل المشروع:

```bash
git clone https://github.com/AbdullahAbuZaid04/abdullah-portfolio.git
```

2. تثبيت المكتبات:

```bash
npm install
```

3. تشغيل المشروع:

```bash
npm start
```
